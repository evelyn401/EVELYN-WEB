# EVELYN PORTFOLIO WEB

個人作品集網站，展示 Evelyn Lin 的專業技能、作品與聯絡資訊。

## 專案簡介

此專案為個人作品集網站，使用現代化的網頁技術建構，包含響應式設計，適配各種裝置尺寸。網站包含個人簡介、技能展示、作品集與聯絡表單等區塊。

## 功能特色

- 📱 響應式設計：支援手機、平板與桌面裝置
- 🎨 輪播圖展示：首頁輪播展示作品
- 🔧 技能卡片：動態載入技能資訊
- 🖼️ 作品集區：展示過往作品
- 📧 聯絡表單：提供訪客聯繫管道
- 🎯 平滑錨點導航：流暢的頁內跳轉體驗

## 專案架構

```
EVELYN WEB/
├── index.html              # 主要 HTML 檔案
├── css/
│   └── styles/
│       └── styles.min.css  # 編譯後的樣式表
├── JavaScript/
│   └── JavaScript.js       # 主要 JavaScript 檔案
└── images/
    ├── Carousel/           # 手機版輪播圖片
    ├── CarouselMD/         # 平板/桌面版輪播圖片
    └── icons/              # 圖示檔案
```

## 技術堆疊

- **HTML5**：語意化標籤結構
- **CSS3/SCSS**：樣式設計與響應式布局
- **JavaScript (ES6 Module)**：互動功能實作
- **Google Fonts**：Noto Sans TC 字型

## 主要區塊

### 1. Header 導航列
- LOGO 與選單按鈕
- 響應式導航選單

- 雙版本輪播圖（手機版 9 張 / 平板版 8 張）

- 自動切換功能

### 3. Profile 個人簡介
- 個人照片展示
- 教育背景與工作經歷
- 專業技能概述

### 4. Skills 技能展示
- 動態載入技能卡片
- 視覺化呈現專業能力

### 5. Work 作品集
- 作品縮圖展示
- 動態載入作品內容

### 6. Contact 聯絡表單
- 姓名、Email、性別欄位
- 訊息文字區域

### 7. Footer 頁尾
- 版權宣告
- 動態年份顯示

## 安裝與執行

### 前置需求
- 現代化瀏覽器（Chrome、Firefox、Safari、Edge）
- 本地伺服器環境（建議使用 Live Server）

### 執行步驟

1. 複製專案到本地端
```bash
git clone <repository-url>
cd "EVELYN WEB"
```

2. 使用 Live Server 或其他本地伺服器開啟 `index.html`

3. 在瀏覽器中開啟 `http://localhost:<port>/index.html`

- Firefox (最新版)
