# EVELYN PORTFOLIO WEB

這是一個使用SCSS建立的個人作品集網站，使用現代化的網頁技術建構，包含響應式設計，適配各種裝置尺寸。網站包含個人簡介、技能展示、作品集與聯絡表單等區塊。

🌐 線上展示
網站連結:https://evelyn401.github.io/EVELYN-WEB/

## 功能特色

- 響應式設計：支援手機、平板與桌面裝置
- 輪播圖展示：首頁輪播展示作品
- 技能卡片：動態載入技能資訊
- 作品集區：展示過往作品
- 聯絡表單：提供訪客聯繫管道
- 平滑錨點導航：流暢的頁內跳轉體驗

## 專案架構

```
EVELYN WEB/
├── index.html              # 主要 HTML 檔案
├── css/
│   └── styles/
│       └── styles.min.css  # 編譯後的樣式表
├── JavaScript/
│   └── JavaScript.js       # 主要 JavaScript 檔案
└── images/
    ├── Carousel/           # 手機版輪播圖片
    ├── CarouselMD/         # 平板/桌面版輪播圖片
    └── icons/              # 圖示檔案
```

## 技術堆疊

- **HTML5**：語意化標籤結構
- **CSS3/SCSS**：樣式設計與響應式布局
- **JavaScript (ES6 Module)**：互動功能實作
- **Google Fonts**：Noto Sans TC 字型
